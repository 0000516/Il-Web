Il Web

The Internet and the World Wide Web are different. The Internet connects computers (e)around the world, while the Web allows people to access websites and share information. The Web was invented in 1989 (nain-tiin ei-ti nain.)by Tim Berners-Lee (Tim berners li) at CERN in Switzerland. He wanted to help scientists share information more easily (ìsili). He created three technologies: HTML for web pages, URLs for web addresses (addressis), and HTTP for sharing information on the Web. Today, millions of people use the Web every day for communication, study and entertainment (entertèinment).

To create the Web, Tim Berners-Lee developed three important technologies:

  • HTML, used to create web pages;
  • URL, the address of a web page;
  • HTTP, the system that allows computers to exchange information on the Web.
  • December 1990: The first website and web server were created at CERN.
  • April 1993: CERN made the Web free for everyone, allowing people to create websites.
  • 1994: Tim Berners-Lee founded the W3C, an organization that helps develop and improve the Web.
